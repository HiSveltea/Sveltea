Instrucciones para publicar en GitHub Pages:
1. Crea un nuevo repositorio en GitHub.
2. Sube el contenido del ZIP (todos los archivos).
3. En la configuración del repositorio, activa GitHub Pages desde la rama 'main' (o 'gh-pages') y la carpeta '/' (root).
4. Tu sitio estará disponible en https://usuario.github.io/repositorio

Pago en línea (nota):
- Este proyecto contiene un flujo de compra simulado (sin procesador real).
- Para integrar pagos reales, recomendamos usar proveedores como Stripe o PayPal:
  - Stripe: integrar Checkout o Payment Intents (requiere clave secreta en servidor).
  - PayPal: integrar Buttons / Checkout (puede hacerse con cliente+servidor).
- No incluyas claves privadas en el frontend.

Personalización:
- Reemplaza las imágenes de productos en /images.
- Edita products.json con tus productos reales (id, name, price, image, description).
- Ajusta estilos en styles.css si deseas otros tonos.
